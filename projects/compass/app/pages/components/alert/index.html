Alerts

Warning Alert

Use this alert type if the user can continue with the action, but they won't be able to revert the changes.

Warning: Check your changes before hitting Save. 

<div class="alert">
    <strong>Warning:</strong> Check your changes before hitting Save.
</div>

 

Danger Alert

Use this alert type if the user has validation errors or other blocking errors that will prevent them from moving forward (e.g. they can't save the form until they fix the validation errors.)

You have 2 validation errors. Please fix them to Save your changes. 

<div class="alert alert-danger">
    You have <strong>2 validation errors</strong>. Please fix them to Save your changes.
</div>

 

Information Alert

Use this alert type to notify the user of a change or update that doesn't require any user intervention.

Note: There are new policy documents available for download. 

<div class="alert alert-info">
    <strong>Note:</strong> There are new policy documents available for download.
</div>

 

Success Alert

Display a success alert on sucessfully saving changes when a user hits Save.

Your changes have been saved. 

<div class="alert alert-success">
    Your changes have been saved.
</div>
