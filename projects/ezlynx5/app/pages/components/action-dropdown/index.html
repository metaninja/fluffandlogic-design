Action Dropdowns

Action Dropdown

When adding an action that applies to the whole screen, or a section of the screen.

Action Dropdowns always appear on the right side of the screen.

Actions
  • Export to Excel
  • Edit Columns
  • Select Transactions to Remove

 


<div class="dropdown">
    <a class="btn btn-primary btn-small dropdown-toggle" data-toggle="dropdown">
        Actions
        <i class="icon-caret-down"></i>
    </a>
    <ul class="dropdown-menu" role="menu">
        <li><a ng-click="getExcelTransactions()">Export to Excel</a></li>
        <li>
            <a ez-column-prefs="" href="javascript:void(0);"><div>
                <div ng-click="openModal()">Edit Columns</div>
            </div></a>
        </li>

        <li>
            <a ng-click="checkTransactionsToDelete()">
                Select Transactions to Remove
            </a>
        </li>
    </ul>
</div>

 

Action Dropdown in a Table Row

When adding an action that applies individually to each table row.

Action dropdowns must appear on the right-most table column.

Actions
  • View/Edit
  • Delete
  • Copy

<div class="btn-group">
    <a class="btn dropdown-toggle btn-link btn-condensed" data-toggle="dropdown" href="#">
        Actions <i class="icon-caret-down"></i>
    </a>
    <ul class="dropdown-menu pull-right" id="commercialMenu@(item.OpenAppID)">
        <li><a title="View/Edit" href="javascript:void(0)" onclick="">View/Edit</a></li>
        <li><a title="Delete" href="javascript:void(0)" onclick="">Delete</a></li>
        <li><a title="Copy" href="javascript:void(0)" onclick="">Copy</a></li>
    </ul>
</div>
