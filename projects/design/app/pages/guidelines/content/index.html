Consistency When Creating Content

To ensure consistency with labels, headings, tables, etc., make sure to follow the rules below when creating content:

Component Should Use * Styling
Form Labels Title Case 12px #555
Radio Buttons Title Case, but can use sentence case with extremely long sentences. 12px #555
Table Heading Row Title Case 12px bold #222
Headings for Editable Form Sections Title Case 12px bold #222
Alerts Sentence Case Varies depending on the alert type
Table Headers UPPER CASE 14px normal #555

 

* In the examples above, this is what we mean by Title Case vs. Sentence Case:

Title Case

This Sentence Is In Title Case. Note That Every Word Is Capitalized, Even Words Like "Is", and "In".

Sentence Case

This sentence is in sentence case. Notice that only the first letter of the sentence is capitalized.
