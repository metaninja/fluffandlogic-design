Badges and Labels

Use this for labeling or categorizations. Badge and label classes must have one parent with the bootstrapForms class. This should happen at the highest reasonable level in the DOM. We should not see a div with this class per badge or label.

Badges

badge
badge badge-success
badge badge-warning
badge badge-important
badge badge-info
badge badge-inverse

Labels

label
label label-success
label label-warning
label label-important
label label-info
label label-inverse

 

(Labels will be exactly the same, just replace badge with label)
