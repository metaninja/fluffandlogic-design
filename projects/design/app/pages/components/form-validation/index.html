Form Validation

  • - If a field is required, highlight the field in red but don't display a tooltip.
  • - If the input is invalid, highlight the field in red using the class below, and display a tooltip.

 


<div class="container-required">
    <input type="text" class="input-validation-error" data-toggle="tooltip"
   data-placement="top" title="Age can only be a number">
</div>
